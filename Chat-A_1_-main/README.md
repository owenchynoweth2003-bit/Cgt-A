# SubShield

SubShield is a React and Vite prototype for a subcontractor insurance compliance vault.

## Active app

The active app is `src/subshield/SubShieldComplete.jsx`.

## Run

npm install
npm run dev

## Build

npm run build

## Main files

- src/App.jsx
- src/subshield/SubShieldComplete.jsx
- src/subshield/data.js
- src/subshield/utils.js
- src/subshield/styles.css
- src/subshield/debug.css

## Notes

The older file at `src/components/SubShield.jsx` is still in the repository as a backup, but it is not used by the active app.
